### ***------------------------------------------- Stored Procedures -------------------------------------------***

### ***Users***
---
1. **sp_createUser()**
    - ***Parametros:***
        - nombre de usuario
        - password
        - email
        - numero de telefono
        - id del Depto
        - id del Municipio
        - direccion
    - ***Retorno:***
        - Bien-> No retorna nada
        - Error-> Retorna error en caso de algun dato incorrecto
        
2. **sp_getUsers()**
    - ***Parametros:***
        - ninguno
    - ***Retorno:***
        - Bien-> Retorna todos los usuarios
        - Error-> Retorna nada

3. **sp_getUserEmail()**
    - ***Parametros:***
        - email
        - password
    - ***Retorno:***
        - Usuario Existe -> Retorna datos del usuario
        - Usuario no Existe-> Retorna 0

4. **sp_getUserID()**
    - ***Parametros:***
        - id del usuario
    - ***Retorno:***
        - Retorna todo el usuario

### ***Filters***
---

15. **sp_searchPublication()**
    - ***Parametros:***
        - Texto que desea buscar
        - Ordenar por fecha
        - Ordenar por precio
    - ***Retorno:***
        - Todas las publicaciones con esa coincidencia

16. **sp_filteredPublication()**
    - ***Parametros:***
        - Categoria -> 'Hogar'
        - Departamento -> 'Atlantida'
        - Municipio -> 'Tela'
        - Rango Inferior -> 0
        - Rango Superior -> 500000
        - Ordenar por fecha -> reciente
        - Ordenar por precio -> caro
    - ***Retorno:***
        - Todas las publicacione que coincidan con los filtros
- **NOTA: En caso de no mandar algun filtro mandar una cadena vacia -> ''**
